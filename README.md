# admin

### 运行项目
```
npm install
npm run s
```

### 命令
```
npm run b // 正式包
npm run t // 测试包
npm run a // 一次打两个包
```

css预处理器：stylus

### 分支
master：主分支

dev：开发分支，日常提交代码

### 规范
tab键：两个空格

引号：单引号
